Hard truth

Finishing this module proves you can run the checklist on paper — it does not auto-certify you. A lead reviews your draft (kill criteria sharp enough to actually kill the bet) and your skill chain plan before you claim a live ticket.

Cert package — what to produce

  1. Assumption — paste your Module 1 one-sentence statement (or refine it).
  2. Draft ticket — six sections with numeric pass/kill in Measure and placeholder Learn.
  3. SOP walkthrough — steps 3–7 written out: labels, column moves, weekly Measure posts, sample Learn decision, Done/next line.
  4. Skill chain plan — for Build: which of /grill-with-docs → /to-spec → /to-tickets → /implement apply, and why; note if you would skip steps for a tiny build.

Keep the draft in a doc, gist, or repo Discussion — do not file or move cards on the live BookIQ board yet.

Failure modes
