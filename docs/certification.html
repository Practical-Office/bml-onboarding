What certification means

Submit when all items are true

Certification is not a badge for reading the course. It means you can create a falsifiable experiment issue, respect board WIP limits, use the Build skill chain when appropriate, and commit to Learn decisions backed by data — including kill when the hypothesis fails.

Your lead initials the checklist (or confirms in your team channel). For BookIQ teammates, graduation to the live board is the usual capstone; other products follow the same checklist with their own board URL in the appendix.

Lead review prompts

Leads can use these questions in a five-minute review:

  1. Can the teammate state the hypothesis in one sentence without reading the issue?
  2. Are pass and kill thresholds numeric and tied to a named measure path?
  3. Does the teammate know when to use /grill-with-docs → /to-spec → /to-tickets → /implement vs skip to /implement?
  4. Will they move the card through Measure without adding scope?
  5. Do they accept kill as a valid Learn outcome?
